Задача №1008. Условие, решение и ответ

Условие

Задание №2

Система наблюдения ежеминутно фиксирует вход и выход посетителей магазина (в минутах, прошедших от начала суток). Считается, что в моменты фиксации входа и выхода посетитель находится в магазине. Нулевая минута соответствует моменту открытия магазина, который работает 24 ч в сутки без перерыва. Менеджер магазина анализирует данные системы наблюдения за прошедшие сутки, и выявляет отрезки времени наибольшей длины, в течение которых число посетителей, находящихся в магазине, не изменялось. Далее менеджер выбирает пики посещаемости — промежутки времени, когда количество посетителей в магазине было наибольшим. Пиков посещаемости в течение суток может быть несколько.

Входной файл содержит время входа и выхода каждого посетителя магазина. Определите, сколько пиков посещаемости было в течение суток, и укажите число посетителей в момент пика посещаемости.

Входные данные

В первой строке входного файла находится натуральное число N (N < 10000) - количество посетителей магазина.
Следующие N строк содержат пары чисел, обозначающих соответственно время входа и время выхода посетителя (все числа натуральные, не превышающие 1440).

Запишите в ответе два натуральных числа: сначала найденное количество пиков посещаемости, а затем число посетителей в момент пика посещаемости.

Типовой пример организации данных во входном файле

6
10 50
100 150
110 155
120 160
130 170
151 170

При таких исходных данных было два пика посещаемости: в отрезки времени со 130 по 150 минуты и со 151 по 155 минуты. Число посетителей в момент пика посещаемости равно 4.


Ответы на задачу:

  1. 2
  2. 643

Решение

file = open('файл.txt') # открываем файл
n = int(file.readline()) # считываем N
izmenenia = [0] * 1441 # создаем список для хранения изменений кол-ва покупателей в каждую минуту
for line in file: # проходимся по строкам файла
    start, end = map(int, line.split()) # считываем 2 числа из строки с помощью map
    izmenenia[start] += 1 # в момент входа кол-во посетителей увеличилось на 1, увеличиваем изменения в эту минуту
    izmenenia[end] -= 1 # в момент выхода кол-во посетителей уменьшилось на 1, уменьшаем изменения в эту минуту
cur_s = 0 # создаем переменную для подсчета текущего количества посетителей
kolvo = [0] * 1441 # создаем список, в который будем записывать сколько посетителей было в магазине в каждую минуту
for i in range(1441): # проходимся по всем минутам
    cur_s += izmenenia[i] # увеличиваем текущее количество посетителей на изменение (если изменение окажется отрицательным, количество уменьшится)
    kolvo[i] = cur_s # записываем в список по индексу данной минуты количество посетителей
m = max(kolvo) # сохраняем максимальное количество посетителей (второй ответ)
pik_cnt = 0 # заводим переменную для подсчета количества пиков
f = True # заводим флаг для проверки, является ли текущее значение новым пиком или продолжением уже подсчитанного
for i in range(1441): # проходимся по всем минутам
    if kolvo[i] == m and f == True: # если в данную минуту количество посетителей равно максимуму и флаг равен истине (новый пик)
        pik_cnt += 1 # то увеличиваем количество пиков на 1
        f = False # меняем флаг, чтобы в случае продолжения пика на следующие минуты снова не подсчитывать
    elif kolvo[i] != m: # иначе если количество посетителей не максимально
        f = True # меняем флаг, если дальше встретится максимальное количество посетителей - это будет новый пик, который надо подсчитать
print(pik_cnt, m) # выводим два ответа: количество пиков и максимальное количество посетителей

Задача №1007. Условие, решение и ответ

Условие

Задание №1

На производстве штучных изделий N деталей должны быть отшлифованы и окрашены. Для каждой детали известно время её шлифовки и время окрашивания. Детали пронумерованы начиная с единицы. Параллельная обработка деталей не предусмотрена.

На ленте транспортёра имеется N мест для каждой из N деталей.

На ленте транспортёра детали располагают по следующему алгоритму:

— все 2N чисел, обозначающих время окрашивания и шлифовки для N деталей, упорядочивают по возрастанию;

— если минимальное число в этом упорядоченном списке — это время шлифовки конкретной детали, то деталь размещают на ленте транспортёра на первое свободное место от её начала;

— если минимальное число — это время окрашивания, то деталь размещают на первое свободное место от конца ленты транспортёра

— если число обозначает время окрашивания или шлифовки уже рассмотренной детали, то его не принимают во внимание.

Этот алгоритм применяется последовательно для размещения всех N деталей.

Определите номер последней детали, для которой будет определено её место на ленте транспортёра, и количество деталей, которые будут отшлифованы до неё.

Входные данные

В первой строке входного файла находится натуральное число N (N < 1000)- количество деталей. Следующие N строк содержат пары чисел, обозначающих соответственно время шлифовки и время окрашивания конкретной детали (все числа натуральные, различные).

Запишите в ответе два натуральных числа: сначала номер последней детали, для которой будет определено её место на ленте транспортёра, затем количество деталей, которые будут отшлифованы до неё.

Типовой пример организации данных во входном файле

5
30 50
100 155
150 170
10 160
120 55

При таких исходных данных порядок расположения деталей на ленте транспортёра следующий: 4, 1, 2, 3, 5. Последней займёт своё место на ленте транспортёра деталь 3. При этом до неё будут отшлифованы три детали.


Ответы на задачу:

  1. 895
  2. 488

Решение

file = open('файл.txt') # открываем файл
n = int(file.readline()) # считываем N
data = [] # создаем пустой список для данных
for i in range(n): # проходимся по индексам строк файла
    shl, okr = map(int, file.readline().split()) # считываем с помощью readline() строку файла, забираем 2 числа - время шлифовки и время окраски
    data.append([shl, 'shl', i + 1]) # добавляем список список из трех элементов - время шлифовки, указываем, что это шллифовка, записываем также номер (он на 1 больше индекса)
    data.append([okr, 'okr', i + 1]) # аналогично действуем с окраской
data.sort() # сортируем все 2N чисел
lenta = [None] * n # создаем ленту конвейера, изначально везде кладем None
polozeni = [] # создаем список для номеров деталей, которые уже размещены на конвейер, чтобы не разместить повторно
for time, tip, i in data: # проходимся по времени, типу и номеру в списке data
    if i not in polozeni: # проверяем, что номера нет среди положенных
        if tip == 'shl': # если тип - шлифовка
            for j in range(n): # то перебираем индексы конвейерной ленты по возрастанию
                if lenta[j] == None: # если место свободно
                    lenta[j] = [i, 'shl'] # кладем деталь: указываем ее номер и тип (шлифовка)
                    polozeni.append(i) # добавляем номер детали в список положенных на конвейер деталей
                    break # останавливаем перебор мест на конвейере, ведь деталь уже размещена
        elif tip == 'okr': # если тип - окраска
            for j in range(n - 1, -1, -1): # то перебираем индексы конвейерной ленты по убыванию (нужно разместить деталь на первое свободное место с конца)
                if lenta[j] == None: # если место свободно
                    lenta[j] = [i, 'okr'] # кладем деталь: указываем ее номер и тип (окраска)
                    polozeni.append(i) # добавляем номер детали в список положенных на конвейер деталей
                    break # останавливаем перебор мест на конвейере, ведь деталь уже размещена
cnt = 0 # заводим переменную для подсчета количества деталей, отшлифованных до последней положенной на ленту
for i in range(n): # проходимся по индексам ленты
    if lenta[i][0] == polozeni[-1]: # если на ленте лежит деталь, номер которой совпадаем с номером последней положенной детали
        break # то останавливаем перебор, нас интересует только шлифовка до последней положенной детали
    elif lenta[i][1] == 'shl': # иначе если тип работы - шлифовка
        cnt += 1 # увеличиваем количество шлифовок на 1
print(polozeni[-1], cnt) # выводим номер последней положенной детали и количество шлифовок до последней положенной детали
